# 《饿魔退散！外卖侠》单局节奏设计

> 系列文档之一。回答你提出的：**单局游戏平均的节奏**。把搜刮、变质、声音/躲避、解障、撤离、结算整合成一条"被感受到"的节奏曲线。
>
> 配套阅读：《地图尺寸与路线规划》（空间）、《饿魔感知与寻人逻辑》《玩家躲避系统》（遭遇）。**单局时长 15~20 min，全局倒计时 15 min（v7.3）。**

---

## 〇、节奏总纲

| 原则 | 说明 |
|------|------|
| 张力来自"来不及"，不是"会死" | 推动节奏的是全局倒计时 + 食物变质，不是某个敌人 |
| 紧→松→更紧 | 搜刮紧张 → 解障专注 → （被迫）应对遭遇 → 带着损失更紧张地撤 |
| 转盘是"非计划性代价" | 转盘不是排好的节奏点，而是"没躲掉"的惩罚；理想局里玩家尽量避免它 |
| 节奏随玩家选择伸缩 | 速攻局 ~5min、稳健局 ~10~12min、贪心局用满 15min |

## 一、代表性单局时间线（"稳健流"为基准）

> 下表为一局典型节奏。实际时长随玩家策略伸缩（见第三节）。

| 时间 | 阶段 | 玩家在做什么 | 张力来源 |
|------|------|-------------|---------|
| 0:00–1:00 | 出发 + 抵达居民区 | 进入、定位、开始第一次搜索 | 低；饿魔尚远 |
| 1:00–4:00 | **居民区搜刮**（低风险） | 搜 3 栋民居，⚪/🟢 食物，可能一次敲冰 | 学习声音系统；第一次"听到声音→降噪/躲"的微时刻；背包食物开始变质 |
| 4:00–4:30 | 推进商业区（决策点①） | "够了就撤 or 继续深入？" | 速攻流在此撤离 |
| 4:30–9:00 | **商业区搜刮**（中风险） | 便利店+餐厅，🟢/🔵 食物，可能钥匙匹配；中噪容器多 | 声音管理吃紧；🔵 食物变质快（~3min）；更多躲避时刻——这是一局的"主菜" |
| 9:00–9:30 | 深区决策点② | "冲深区拿🟣/隐藏房间 or 现在撤（食物在变质、倒计时过半）？" | 核心取舍最重时刻 |
| 9:30–13:00 | **深区**（高风险） | 渡电、隐藏房间（🟣/传说）；🟣 食物 ~90s 变质 | 峰值紧张；操作不慎易被锁定→转盘（代价） |
| 13:00–15:00 | **撤离 race** | 背着变质食物长途返程；**15:00 硬线·未撤=迷失失全部背包**；撤离点 10s 倒计时 | 全局最高潮——生死线 |
| 结算 | 合作结算 | 品质×完成度 + "差一点"展示 | 释放 + "再来一局"冲动 |

## 二、张力曲线

```text
张力
 ▲                                                   ╱╲  ← 撤离race(峰值)
 │                                          深区 ╱╲ ╱  ╲
 │                                  决策点② ╱      ╲    ╲
 │                    商业区(主菜) ╱╲  ╱╲ ╱                ╲___ 结算(释放)
 │            居民区 ╱╲      ╱╲ ╱    ╲╱                          → "再来一局"
 │   出发 ___╱        ╲____╱
 └────────────────────────────────────────────────────────────▶ 时间
   0min    1      4        9          13              15min   结算

  ⚡尖刺(随机出现) = 被惊动→躲；若没躲掉→转盘(90s)把曲线顶到顶并造成损失
```

- 整体是**缓升 → 商业区起伏 → 深区抬升 → 撤离峰值 → 结算释放**。
- **转盘**不在曲线的固定位置：它是随机尖刺，理想局里玩家用"躲"把尖刺压平；一旦没躲掉，90s 转盘把张力顶满，并让**后续时间更少、食物更少**，从而抬高之后整段曲线。

## 三、节奏随策略伸缩

| 策略 | 节奏形态 | 大致时长 |
|------|---------|---------|
| 速攻流 | 居民区搜完即撤，短促、低峰 | ~5 min |
| 稳健流 | 居民+商业，完整起伏 + 一次中等撤离race | ~10~12 min |
| 贪心流 | 冲深区拿🟣/传说，用满倒计时，峰值最高、返程race最险 | ~15 min |

地图与数值要让三者**都成立但风险/回报不同**（见《地图尺寸与路线规划》§4.2）。第 1 局玩家多走速攻，第 3 局多走贪心——这本身就是"3 局验证"的节奏进化。

## 四、各系统在节奏中"何时咬人"

| 系统 | 咬人时刻 |
|------|---------|
| 食物变质 | 全程隐形计时器；🔵 在商业区开始逼人（~3min），🟣 在深区极限逼人（~90s） |
| 全局倒计时 | 过半（~7:30）触发"该不该再深入"的焦虑；**15:00 硬上限·未撤=迷失失全部背包**——撤离时机是核心赌注 |
| 声音/感知 | 每次高噪动作（敲冰/渡电/奔跑）都可能招饿魔；商业区/深区最频繁 |
| 躲避 | 遭遇时的"降噪/躲"微节奏；理想是 15~25s 化解，远短于转盘 |
| 解障 | 主动选择的"专注插曲"：敲冰 8~12s、钥匙 25~45s、渡电首名 15~25s |
| 饿魔难度递增 | 随单局时间 T1→T3 抬升：追人越来越快、被抓后转盘越来越刁钻 → 后段"躲"更难、被抓更肉痛，与变质/倒计时叠加把后半局张力整体抬高 |
| 转盘（代价） | 非计划性；出现即 90s 大尖刺 + 食物损失，抬高后续张力；**越晚被抓难度档越高**（T1/T2/T3） |
| 结算 | 节奏收束；"差一点"制造续局冲动 |

## 五、节奏设计的两条红线

1. **不要让转盘变成"想要的节奏点"**。它是惩罚。原"完胜吐出额外奖励"会诱导故意被抓 → **已改为条件式奖励**：奖励只给"饱食满 + 全程零怒气"的零瑕疵表演，且随时间难度递增越来越难拿，被抓的现实最优几乎都是"白花 90s"的零损失（见《可配置参数总表 v1》§13.3）。转盘始终是玩家想避免的尖刺。
2. **不要让"躲"把节奏拖平**。靠全局倒计时 + 变质保证"躲太多 = 食物凉了/没搜够"（见《玩家躲避系统》§5），让缓段始终有"在亏钱"的暗流，节奏松而不断。

## 六、灰盒验证清单
1. 玩家是否在 ~9min 处真的纠结"撤还是冲深区"？
2. 撤离 race 是否是全局最紧张的时刻（背着变质食物 + 倒计时将尽）？
3. 三种策略的节奏是否都成立、且体验明显不同？
4. 没出现转盘的"干净一局"是否依然紧张（靠时间+变质+躲）？
5. 出现转盘后，玩家是否感到"亏了、后面更紧了"，而不是"赚了一把小游戏"？
