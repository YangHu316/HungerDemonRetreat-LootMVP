# 🔌 小游戏接入大地图 & 开发者模式 —— 程序说明

> 适用：`ice_cabinet`(🧊敲冰) / `electric_crossing`(⚡渡电) / `gluttony_portal`(🌀暴食) 三个小游戏场景。
> 本轮把它们从「独立测试场景」改造成**可直接实例化进大地图的组件**；把测试快捷键统一收到一个开发者模式开关后；并为美术资产预留替换槽（不再全脚本写死）。

---

## 一、作为组件实例化进大地图
每个小游戏根脚本都暴露统一接口：

| 成员 | 说明 |
|------|------|
| `@export var standalone: bool`（默认 **false**） | **false（接大地图）**：`_ready` 只搭几何体/面板，**不建相机/环境/光照、不自动开始**，用宿主相机取景/拾取。**true（沙盒调试）**：自带相机+环境+光照并自动 `begin()`（仅 dev_launcher 用）。 |
| `func begin()` | 开始一局（玩家交互触发时由宿主调用）。 |
| `signal finished(result: Dictionary)` | 一局结束发出 → 主循环据此收回控制、发奖。 |

`finished` 的 result：
- 🧊 敲冰：`{won, food, rarity}`（取出的食物）
- ⚡ 渡电：`{crossed: true}`（过池成功，可让队友通行）
- 🌀 暴食：胜利 `{won:true, loot:[名称…]}`；撕裂抢救结束 `{won:false, fled, bagged_value, bagged_count}`

**接入示例（伪码）**：
```gdscript
var mg = preload("res://ice_cabinet.tscn").instantiate()
mg.position = 道具世界坐标          # 3D：整组几何体随根节点变换，可摆到地图任意处
mg.finished.connect(_on_minigame_done)
add_child(mg)
# 玩家交互时：主相机移到取景位 → mg.begin()
```
- 3D 小游戏（敲冰/渡电）几何体都相对**自身原点**搭建，挪根节点即整体就位。
- 暴食是 **2D UI 面板**：`begin()` 前不绘制（`_started=false`），宿主在交互时显示它。
- 震屏 / FOV punch 等「相机特效」接大地图时**不动宿主相机**（脚本已 null 守护）——交给主循环统一处理。

## 二、开发者模式（统一快捷键开关）
- 全局 autoload **`Dev`**（`dev_mode.gd`，已在 project.godot 注册）：按 **F1** 开/关；开启时右上角显示「● 开发者模式 ON」。
- **测试快捷键只在开发者模式下生效**：敲冰 `1/2/3` 换柜大小、三者的 `R` 重置。正式版玩家默认碰不到。
- 正常玩法操作（鼠标砸/擦、空格蓄力、方块移动旋转、抢救点击 等）**不受开关影响、始终可用**。

## 三、美术资产替换槽（留空=灰盒占位；大部分是 3D 模型）
编辑器选中小游戏根节点 → Inspector「美术替换槽」填入资源即可替换占位灰盒：

| 小游戏 | 槽位（类型） |
|--------|------|
| 🧊 敲冰 | `cabinet_model`(PackedScene 冰柜外壳) · `food_models`(Dictionary 食物名→PackedScene) · `shard_model`(PackedScene 碎片) |
| ⚡ 渡电 | `stone_model` · `bank_model` · `player_model` · `follower_model` · `switch_base_model` · `switch_lever_model`(会绕底座转) · `water_material`(Material) |
| 🌀 暴食(2D) | `food_textures`(Dictionary 稀有度int→Texture2D) · `portal_texture`(Texture2D) |

- 槽位**留空 → 脚本生成灰盒占位**（现状）；**填了 → 用美术资源**。
- 约定：3D 模型以「自身原点 = 放置锚点」；敲冰外壳模型以**正面玻璃门中心**为锚。
- 注意：渡电 `water_material`/闸刀模型、敲冰用食物模型时，脚本的**程序化变色**（水电光脉动、闸刀红↔绿、食物变质变暗）会**让位给美术资源**——这些状态表现需美术/shader 自行处理（脚本已留接口、不再强制改色/改 modulate）。

## 四、F5 开发沙盒
- 主场景 = **`dev_launcher.tscn`**：实例化一个小游戏(standalone=true)单独试玩。
- **F2** = 切换下一个小游戏；**F1** = 开发者模式（启用测试快捷键）。
- **接大地图时不要用 dev_launcher**，直接实例化三个小游戏 `.tscn`（standalone 留默认 false）。
