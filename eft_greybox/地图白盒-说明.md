# 🗺️ 测试关卡白盒 v3（5分钟·单向搜打撤）— 说明

> **`map_greybox.tscn`**：一张**俯视角搜打撤**测试关卡白盒，目标**单局~5分钟、一条不回头的撤离路线、展示大部分玩法系统**。参照乡村鸟瞰图(村屋/农田/桥/溪/窝棚)，全部 **CSG 独立组件**(非脚本)。设计由多方案workflow综合而来。
> 地图 X∈[-124,150] Z∈[-200,200]，俯视：X横(图左-X/右+X)，Z纵(图上-Z/下+Z)，地面 y=0，米。

## 怎么跑
- **F5**：主场景＝`map_greybox.tscn`，自带正俯视概览相机看全貌。回小游戏沙盒：`project.godot` 的 `run/main_scene` 改回 `dev_launcher.tscn`。
- 编辑器双击场景自由查看；每个组件可单独点选/移动/删除/替换。

## 单向路线（出生A → 撤离B，全程不回头 · 7段·约5分钟）
| # | 节点(世界坐标) | 动作 | 展示系统 | ~秒 |
|---|------|------|---------|----|
| 1 | 出生·院落纸箱 (-43,40) | 潜行搜纸箱/垃圾桶，体会 静步1.5 vs 走5 vs 搜4~6m | 出生 + 场景交互 + 声音 | 35 |
| 2 | 长屋搜刮+敲冰 (-55,-76) | 进长屋搜货架(**钥匙#1**)；角落**敲冰冰柜**(多格盲盒·砸冰 5~6m持续+终破10m) | 场景交互 + 🧊敲冰 + 声音(高噪) | 60 |
| 3 | 听声→降噪躲避 (-46,-88) | 敲冰惊动饿魔→它扑向**声源点**(非你)；你静步离开/钻储物柜，~15~25s等它放弃 | 敌人(扑声源·放弃) + 躲藏 | 20 |
| 4 | 白棚补钥匙 (-60,-150) | 北上白棚搜货架(**钥匙#2**)；可选角落暴食传送门(高风险·可跳) | 场景交互 + 🌀暴食(可选) | 30 |
| 5 | 横穿开阔农田 (48,-50) | 贴田垄/钻窝棚草垛过最暴露段；菜窖冰柜搜(**钥匙#3**·开阔发声)；饿魔C线~2:30必经一次 | 声音(暴露vs掩体) + 敌人巡逻 + 躲藏 + 🧊敲冰 | 75 |
| 6 | **凌波渡电过溪** (108,-118) | 蓄力跳5颗落脚点到东岸→拉电闸断该段电；落水15m+弹回。渡过即过单向分界(西岸被电墙切断) | ⚡渡电 + 声音(15m最大) + **单向闸1** | 40 |
| 7 | 开门撤离 (124→138,-118) | 三钥匙开**撤离大门**(开门8~10m高噪·门旁窝棚躲)→踩撤离台 10s 倒计时 | 🔑钥匙与大门 + 声音 + 撤离 + **单向闸2** | 35 |

**5分钟配速**：走55 + 搜70 + 小游戏90 + 躲35 + 其它45 = **~295s**。

## 五大区(俯视可读·颜色图例)
- **① 村屋区(西)**：深色长屋(可进·搜刮+敲冰+钥匙#1) + 白色大棚(钥匙#2+暴食) + 院落停车坪 + 小民居。**出生在此**。
- **② 开阔农田(中右)**：绿地+田垄(半掩体)+草垛/窝棚/过渡小屋(全掩体)+菜窖冰柜(钥匙#3)。**饿魔高压潜行段**。
- **③ 通电溪流(东缘·蓝发光)**：电墙·南北封到墙·不可涉；**唯一渡口**(落脚点+电闸)=单向闸1。
- **④ 东岸撤离区(东北)**：三钥匙大门=单向闸2 → 橙色撤离台。桥=饿魔C线巡逻路(玩家走渡电·勿走桥)。
- **颜色**：深灰=屋 / 白=大棚 / 绿=田·暗绿=田垄 / 蓝发光=电溪·蓝=窝棚 / 棕=桥·灰=路 / 黄=钥匙/电闸 / 青=敲冰 / 紫=暴食 / 红=饿魔锚 / 绿台=出生 / 橙台=撤离 / 青绿=躲点。

## 系统落点核对
- **声音**：分级地标——出生纸箱4m/垃圾桶6m → 长屋敲冰5~6m+终破10m(封闭=易听见) → 农田搜8m+跑12m(开阔=暴露) → 渡电落水15m(最大) → 开门8~10m。开阔农田 vs 封闭长屋 = 可读的"高暴露 vs 有掩体"对比。
- **敌人交互**：唯一饿魔·四态(巡逻→调查声源→搜寻→锁定)；A线(村屋·红锚-50,-30)+C线(全图含桥·红锚55,-60)。核心桥段=敲冰惊动→扑声源→钻柜→15~25s放弃(<<被抓代价)。农田~2:30脚本必经一次。**多玩家追最近**(见《感知》§2.5)。
- **场景交互**：纸箱/货架/垃圾桶/菜窖遍布，产食物+3钥匙(每区一把保底·全在溪西防死锁)。
- **小游戏**：🧊敲冰(长屋+菜窖)、⚡渡电(过溪·单向闸)、🔑钥匙与大门(撤离闸)、🌀暴食(白棚角·可选)。
- **出生/撤离**：村屋(A·-43,40) → 东岸外部路(B·138,-118)，两端不同，单程~200m。
- **单向**：通电溪流(X108)南北封到墙=电墙；仅渡口可渡，拉闸后西岸断电墙切断=不能涉回；+三钥匙门终封。钥匙全在溪西、东岸无钥匙=不死锁。桥是饿魔的路、玩家不走=回头既险又无意义。

## 组件化 & 取舍
- 全部独立 CSG 组件(`CSGBox3D`/建筑用 `CSGCombiner3D` 空心壳)；结构/掩体/容器已开 `use_collision`，加角色直接能走、被挡、可躲。
- **两栋主搜刮屋(长屋/白棚)做成开顶空心壳**(俯视可看进去、容器在内可搜)；其余建筑=实心平顶massing做掩体。
- 容器/小游戏/钥匙/电闸=彩色小方块标记(白盒占位)。空旷农田已用过渡小屋/草垛/窝棚/树补掩体与变数。
- **这是一张 one-way 教学测试关**(出生≠撤离)，为展示单向特意偏离"入口=出口·返程取舍"的正式设定；正式关卡仍回入口。
